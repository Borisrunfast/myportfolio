Holidaze App ✨

Responsive accommodation booking platform with full user registration, multi-role management, and calendar-based availability. Built during the FED2 exam to showcase planning, design, and technical skills.

View Live Site Source Code

Key Features & User Stories

✔

Visitor: browse and search venues, view availability calendar with booked dates highlighted.

✔

Customer: register/login, create and manage bookings, update profile avatar.

✔

Venue Manager: register/login, create/edit/delete venues, view bookings, update avatar.

✔

Admin capabilities with role-based permissions and calendar integration.
